Vacuum cleaner robot sim
========================

Scala 2.11.4

Sbt 0.13.6

Scala reactive:
  - scalareactive.org
  - https://github.com/nafg/reactive


Starting
========

Start with default tick of 1000 ms:
sbt> run

Start with non-delayed ticks:
sbt> run 0

Start with tick of 3000 ms:
sbt> run 3000
